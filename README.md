## Yaroslav Hayduk Portfolio

Single-page portfolio built with Next.js App Router, TypeScript, Tailwind CSS, and reusable component architecture.

### Run locally

```bash
npm install
npm run dev
```

### Content editing

All portfolio copy and translation content lives in:

- `src/data/portfolio-data.ts`

Sections and UI are split into reusable components under:

- `src/components/layout`
- `src/components/sections`
- `src/components/ui`
